MDViewer

A minimal macOS Markdown viewer. No editor, no bloat — just clean rendering with Dark Mode.

Download View on GitHub

GitHub-style Rendering

Markdown rendered with marked.js — tables, code blocks, task lists, all looking clean.

Dark Mode

Follows system appearance automatically, or switch manually via the View menu.

Native macOS

Built with SwiftUI. Open files, Recent Files, drag & drop — it just works.

Lightweight

Single-purpose app. No Electron, no dependencies, no account required.

Build from Source

brew install xcodegen
xcodegen generate
xcodebuild -scheme MDViewer -configuration Release build
